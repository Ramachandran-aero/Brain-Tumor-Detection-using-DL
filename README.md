# Brain-Tumor-Detection-using-DL
Brain Tumor Detection Using Deep Learning

**Project Overview**

The Brain Tumor Detection Using Deep Learning project is designed to automatically identify whether a brain MRI image contains a tumor or no tumor using Deep Learning and Computer Vision techniques.

This project demonstrates the application of Artificial Intelligence in healthcare, where medical image analysis can help support early diagnosis and improve treatment planning.

**Objective**

The main objective of this project is to:
Analyze MRI brain scan images
Detect the presence of a brain tumor
Build a deep learning model for medical image classification
Assist in faster and more accurate diagnosis using AI

**Tech Stack**
Python
TensorFlow / Keras
NumPy
Pandas
Matplotlib
Seaborn
OpenCV
Jupyter Notebook

**Dataset Description**

The dataset consists of brain MRI images categorized into:

Tumor
No Tumor

The images are preprocessed and used to train a deep learning model for binary image classification.

**Project Workflow**

The project follows a structured deep learning pipeline:

Data Collection
Load MRI image dataset
Organize images into tumor / no tumor classes
Image Preprocessing
Resize images
Normalize pixel values
Convert images into suitable array format
Exploratory Visualization
Visualize sample MRI images
Understand image class distribution
Model Building
Build a deep learning / CNN-based model
Train the model on MRI images
Model Evaluation
Accuracy and Loss analysis
Prediction performance on test images
Prediction
Classify MRI scans as Tumor or No Tumor

**Deep Learning Model**

A Convolutional Neural Network (CNN) is used in this project to automatically learn image features and classify MRI scans.

The model helps in:

Extracting important visual patterns from MRI images
Differentiating healthy scans from tumor-affected scans
Supporting automated medical image diagnosis

**Key Insights**

Some important observations from this project include:

Deep learning can effectively identify patterns in MRI images
Proper image preprocessing significantly improves model performance
CNN models are highly suitable for medical image classification tasks
AI can assist doctors by acting as a decision-support tool

**Results**

The model was able to classify MRI scans into Tumor and No Tumor categories effectively, demonstrating the usefulness of deep learning in healthcare applications.

This project can help in:

Supporting early tumor detection
Reducing manual image screening effort
Improving diagnostic efficiency
Showcasing AI applications in medical imaging
